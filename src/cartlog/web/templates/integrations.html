Integrations

Send receipts with Apple Shortcuts

Install the cartlog Shortcut to send a receipt image or PDF straight from the share sheet on iPhone, iPad, or Mac. No app to install, and cartlog only needs to be reachable from the device.

Install the Shortcut

When you add it, the Shortcut asks once for your cartlog URL. Use:

{{ upload_url }}

Authentication

The Shortcut must include an API token with every upload request. cartlog checks for the token in either of these HTTP headers - include exactly one:

Header Example value
X-Cartlog-Token your-api-token
Authorization Bearer your-api-token

To create a token, go to Account › API Tokens. You need an Editor or Admin role. The token is shown only once at creation - copy it before closing the dialog.
